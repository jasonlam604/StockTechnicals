## Requirements

* Maven 3.x
* JUnit 4
* TravisCI

## Indicators

* ADX - Average Directional Index
* CCI - Commodity Channel Index
* EMA - Exponential Strength Index
* MACD - Moving Average Convergence/Divergence Oscillator
* PSAR - Parabolic SAR
* RSI - Relative Strength Index
* SMA - Simple Moving Average
* TP - Typical Price
